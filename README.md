## 😃 关于我们 
**这里是中国Web3协会，成立于2023年2月7日**<br>  
（也许是，也希望未来是）中国最早的Web3社群<br>  
我们建立的初衷是因为创始人烟波,苦于中文互联网中的 Web3 相关知识分享内容相对较少，而为了改变这一现状，希望未来能整合和拓宽这类方向相关的 中文知识内容，从而建立了该组织。<br>
